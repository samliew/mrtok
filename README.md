# MRTOK - MRT Okay or Not?

[![License](http://img.shields.io/badge/License-MIT-blue.svg)](http://opensource.org/licenses/MIT)


## About

MRTOK displays Singapore MRT trains' real-time statuses from official sources and social media.

This repo contains the client-side JavaScript code that does all the processing logic on the user's web browser.

**FAQ:** [https://mrtok.com/faq](https://mrtok.com/faq)
